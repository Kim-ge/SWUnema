      3. 큐레이션

      - 왼쪽에는 가장 큰 포스터가 위치하고 아래에 상세정보와 예매버튼 구현

      4. 메가박스 안내

      1. header

      -상단 메뉴에 마우스를 올리면 하위메뉴가 나타나도록 구현

      -사이트맵, 로그인 아이콘을 클릭하면 하단에 숨겨져있던 레이어 등장

      -검색 아이콘 영역 : 각 영화의 이름 위에 마우스를 올리면 포스터가 바뀜


      2. container (영화리스트)

      - 각 영화 포스터 위에 마우스를 올리면 영화 줄거리 및 관람평이 보임


      3. 페이지 스크롤 시

      -페이지를 하단으로 내리면 상단에 페이지의 현재위치 표시

      -오른쪽 하단에 top버튼을 누르면 페이지 상단으로 이동

    • P3 상세설명 보기

      • 담당 : 데이터사이언스학과 2021111712학번 김고은



      1. header 영역

      - a태그 요소 : 마우스 hover시 text-underline, 해당페이지 링크 연결

      - 상단 드롭다운 메뉴 : css hover 이벤트를 주어 하위메뉴가 펼쳐지도록 구성, 자바스크립트를 활용한 border이벤트

      - 페이지 레이어 : 자바스크립트 함수 활용, 아이콘 클릭시 레이어 등장 바뀐 아이콘 다시한번 클릭시 레이어 숨김

      - 검색창 레이어 : input태그를 사용해 검색어 입력 기능 구현


      2. container iframe 영역

      - 자바스크립트 함수 활용, 상단 날짜 부분에 마우스 올리면 하단 border등장

      - 전체영화 리스트 : jquery라이브러리 mCustomScrollbar 활용, 스크롤 기능 구현


      3. footer 영역

      - a태그 요소 및 sns 아이콘 : 마우스 hover시 text-underline, 해당페이지 링크 연결

      - 극장찾기 버튼 : 극장찾기 레이어 삽입, 닫기아이콘에 자바스크립트 함수 걸어 레이어닫기 기능 구현
